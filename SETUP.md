# Setup Instructions

## Step 1: Create GitHub Personal Access Token

1. Go to: https://github.com/settings/tokens?type=beta (Fine-grained tokens)
2. Click **"Generate new token"**
3. Configure:
   - **Name**: `metrics-token`
   - **Expiration**: 90 days (or longer)
   - **Repository access**: Select "All repositories"
   - **Permissions**:
     - `Contents`: Read and write
     - `Metadata`: Read-only
4. Click **"Generate token"**
5. **COPY THE TOKEN** (you won't see it again!)

---

## Step 2: Add Token to Repository Secrets

1. Go to your profile repo: https://github.com/DrZuzzjen/DrZuzzjen
2. Click **Settings** → **Secrets and variables** → **Actions**
3. Click **"New repository secret"**
4. Configure:
   - **Name**: `METRICS_TOKEN`
   - **Secret**: Paste your token from Step 1
5. Click **"Add secret"**

---

## Step 3: Push Files to Your Repo

### Option A: If repo already exists

```bash
# Clone your existing repo
git clone https://github.com/DrZuzzjen/DrZuzzjen.git
cd DrZuzzjen

# Copy the files from github-tracker
cp -r ../github-tracker/.github .
cp ../github-tracker/README.md .

# Push changes
git add .
git commit -m "Add GitHub metrics and snake animation"
git push origin main
```

### Option B: Fresh start

```bash
# Navigate to the project folder
cd E:/code/github-tracker

# Initialize git
git init
git add .
git commit -m "Initial commit: GitHub profile with metrics"

# Connect to your repo and push
git remote add origin https://github.com/DrZuzzjen/DrZuzzjen.git
git branch -M main
git push -u origin main --force
```

---

## Step 4: Run the Workflow

1. Go to: https://github.com/DrZuzzjen/DrZuzzjen/actions
2. Click on **"GitHub Metrics"** workflow
3. Click **"Run workflow"** → **"Run workflow"**
4. Wait ~2-3 minutes for it to complete

---

## Step 5: Verify

Visit your profile: https://github.com/DrZuzzjen

You should see:
- Your new README with metrics SVG
- The snake animation (in the `output` branch)

---

## Customization

### Change metrics plugins

Edit `.github/workflows/metrics.yml` and modify the plugins.

Full documentation: https://github.com/lowlighter/metrics/blob/master/README.md

### Available plugins you might want to add:

```yaml
# Spotify currently playing
plugin_music: yes
plugin_music_provider: spotify
plugin_music_token: ${{ secrets.SPOTIFY_TOKEN }}

# WakaTime coding stats
plugin_wakatime: yes
plugin_wakatime_token: ${{ secrets.WAKATIME_TOKEN }}

# Star history chart
plugin_stargazers: yes
plugin_stargazers_charts_type: classic

# Topics/interests
plugin_topics: yes
plugin_topics_mode: starred
plugin_topics_limit: 15
```

## Troubleshooting

### Metrics not generating?
- Check Actions tab for errors
- Verify `METRICS_TOKEN` secret is set correctly
- Token needs `repo` scope

### Snake not showing?
- Wait for the snake workflow to create the `output` branch
- Check if images exist at: `https://github.com/DrZuzzjen/DrZuzzjen/blob/output/github-snake.svg`
